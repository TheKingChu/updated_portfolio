About

One More Door is a puzzle game made for andriod. Solve small puzzles in each room to find the key and unlock the door to get to the next. Can you complete every puzzle?

Key Features:

  • Phone Inputs: Tilt, shake and more to solve puzzles
  • Different Themes: 8 puzzles for each theme
